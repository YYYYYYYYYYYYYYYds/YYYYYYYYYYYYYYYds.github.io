市值参数管理网页使用说明
=======================

📁 文件结构：
- index.html   - 主页面文件
- style.css    - 样式文件 (Win7风格)
- script.js    - JavaScript功能文件
- README.txt   - 说明文档

🎯 功能：管理智能合约的市值参数

文件结构说明：
---------

📄 index.html
   - 网页主结构，引用外部CSS和JS文件
   - 包含钱包状态、统计信息、参数表单等UI组件

🎨 style.css  
   - Win7经典风格样式
   - 响应式布局，移动端友好
   - 包含按钮动效、加载动画等

💻 script.js
   - Web3钱包连接逻辑
   - 合约交互功能 (getAllParams, setMarketParams)
   - 表单验证和数据处理

使用步骤：
---------

1. 配置合约地址
   打开 script.js 文件，找到第2行的配置：
   
   const CONTRACT_CONFIG = {
       test: true, // 改为false使用主网
       testnet: {
           contractAddress: '在这里填入测试网合约地址',
           ...
       },
       mainnet: {
           contractAddress: '在这里填入主网合约地址',
           ...
       }
   };

2. 使用方法
   - 用浏览器打开 index.html
   - 页面会自动尝试连接钱包（需要安装MetaMask）
   - 连接成功后会自动读取合约参数
   - 修改参数后点击"更新参数"按钮

功能说明：
---------

📊 显示统计信息：
   - 买入花费USDT：累计买入消耗的USDT数量
   - 买入得到代币：累计买入获得的代币数量
   - 卖出得到USDT：累计卖出获得的USDT数量
   - 卖出代币数量：累计卖出的代币数量
   - 当前价格：实时价格
   - Gas消耗：累计gas费用价值

⚙️ 可修改参数：
   - 买入金额范围：每次买入的USDT金额范围
   - 卖出金额范围：每次卖出的代币数量范围
   - 价格区间：触发买卖的价格区间
   - 时间间隔：操作之间的等待时间

注意事项：
---------

⚠️ 权限要求：
   - 修改参数需要JJ权限（onlyJJ修饰符）
   - 确保连接的钱包地址有相应权限

🔧 网络设置：
   - 确保钱包连接到正确的网络（BSC主网或测试网）
   - 根据需要修改 test 参数切换网络

💰 Gas费用：
   - 更新参数需要支付BNB作为gas费
   - 建议钱包保持一定的BNB余额

🎨 界面特点：
   - Win7经典风格设计
   - 移动端友好响应式布局
   - 实时状态反馈和错误提示

代码结构优势：
---------

✅ 模块化设计：
   - HTML、CSS、JS分离，便于维护
   - 样式和逻辑独立，提高代码可读性

✅ 性能优化：
   - 外部CSS和JS文件可被浏览器缓存
   - 减少页面加载时间

✅ 易于扩展：
   - 样式修改只需编辑CSS文件
   - 功能增加只需编辑JS文件
   - HTML结构相对稳定

技术说明：
---------

📡 合约交互：
   - 读取：getAllParams() - 获取所有参数
   - 写入：setMarketParams() - 更新市场参数

🔗 依赖：
   - Web3.js：区块链交互库 (CDN加载)
   - MetaMask：钱包连接

📁 文件关系：
   index.html ← style.css (样式)
             ← script.js (功能)
             ← Web3.js (CDN)
